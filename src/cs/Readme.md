## Основы

### Типы данных

```c#
bool    - System.Boolean
byte    - System.Byte    (1 byte)
sbyte   - System.SByte   (1 byte)
short   - System.Int16   (2 bytes)
ushort  - System.UInt16  (2 bytes)
int     - System.Int32   (4 bytes) int a = 5
uint    - System.UInt32  (4 bytes) uint a = 5u
long    - System.Int64   (8 bytes) long a = 5l
ulong   - System.UInt64  (4 bytes) ulong a = 5ul
float   - System.Single  (4 bytes) float a = 3.14f
double  - System.Double  (8 bytes) double a = 3.14
decimal - System.Decimal (16 bytes) decimal a = 3.14m
char    - System.Char    (2 bytes)
string  - System.String  (набор символов Unicode)
object  - System.Object  (4-8 bytes 32x-64x)
```
